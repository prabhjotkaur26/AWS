# CLOUD
“Cloud” (or cloud computing) is a way of using computers, storage, and software over the internet instead of having everything on your own device.

## Simple Idea

Imagine you have photos, videos, or apps. Instead of saving them only on your phone or computer, you store and use them on powerful computers somewhere else (called servers) that you access through the internet.

That “somewhere else” is the cloud.

##  Easy Example
When you use Google Drive → your files are stored in the cloud
When you watch movies on Netflix → videos are coming from the cloud
When you send emails using Gmail → everything is handled in the cloud

## How It Works (Step-by-Step)
You connect to the internet
You open an app or website
Your request goes to remote servers (data centers)
These servers process your request
Results come back to your device

You don’t see the servers—they are spread across the world.

## Types of Cloud Services
### 1. Storage Cloud
Saves files online
Example: photos, documents

### 2. Software Cloud (SaaS)
Use apps without installing them
Example: online Word, email

### 3. Platform Cloud (PaaS)
Helps developers build apps

### 4. Infrastructure Cloud (IaaS)
Provides virtual computers and servers

## Real-Life Uses
1.Backup your phone data
2.Online gaming
3.Video streaming
4.Online classes
5.Business apps and websites

## Advantages
1.Access from anywhere
2.No need for big storage on your device
3.Automatic updates
4.Easy sharing

## Disadvantages
1.Needs internet connection
2.Privacy concerns
3.Can cost money for extra storage
